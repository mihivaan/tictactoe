# Tic Tac Toe Game

A simple browser-based Tic Tac Toe game using vanilla JavaScript and DOM manipulation.

## Features
- 2-player mode
- Win/tie logic
- Replayable game board

## Tech Stack
- HTML/CSS
- JavaScript (DOM)

## How to Play
1. Open the game in a browser
2. Click on cells to take turns
3. Winner is displayed
